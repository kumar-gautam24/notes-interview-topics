# EMERGENCY PROTOCOLS

## If You Miss a Day:
```
1. Don't panic
2. Don't try to "catch up" by doubling next day
3. Just do today's work
4. Note in journal: Why did I miss? How to prevent?
5. Get back on track immediately
```

## If You Want to Quit (You Will Feel This):
```
Expected times:
- Week 3: Novelty wears off
- Week 8: Progress feels slow
- Week 16: Plateau phase
- Week 25: Interview rejections hurt

When you feel this:
1. Read your Day 1 notebook entry
2. Review your progress (you've come far!)
3. Talk to accountability partner
4. Take 1 rest day (not 3!)
5. Remember: Quitting guarantees failure. Continuing gives you a chance.
```

## If You Get Rejected:
```
Normal. Expected. Part of the process.

After rejection:
1. Ask for feedback (they probably won't give, but try)
2. Identify what went wrong
3. Improve that specific area
4. Apply to 5 more companies
5. Keep going

50 applications = 5 interviews = 1 offer (industry average)
```

## If You Get Sick:
```
Minimum mode:
- Take your supplements
- Do 1 DSA problem (15 min)
- Commit to GitHub (keeps streak)
- Rest

Don't try to study 2 hours when sick.
Better to do 15 min than 0 min.
```
